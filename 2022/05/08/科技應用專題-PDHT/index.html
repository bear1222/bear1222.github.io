科技應用專題-PDHT

Author: n0 Bear Date: May 8, 2022  14:23:04 Category: 課程學習成果

簡述

帕金森氏症患者目前僅能透過醫生、家人目測進行篩檢，不只容易有誤差也難以進行比較。因此我們結合Arduino、App Inventor、PHP、JavaScript製作了此裝置，不僅將測量結果量化，也能將數據視覺化作圖，讓醫生能依據患者狀況，做出更適合的診斷。

心得

這次專題中,我印象最深刻的,是在量化手指開合測驗的數據時,最直觀的方法當然是透過直接測量拇指與食指距離來實現,但因為種種限制,最後使用的方式,是讓食指移動時推動障礙物,超音波感測器用來測量它與障礙物間距離。因此我學到了在達到目的的過程中,若遇到一些阻礙,有時候可以不用直接跟它硬碰硬,可以試著通過其他方法來達成。

另外,因為這是第二次製作專題了,我覺得我在寫 code 時也有了一些進步,像是有些簡單但很冗長的 code,我會把它包裝在一個 function 裡,這樣不只讓我的 code 變得較簡潔易理解,要 debug 時也較容易。

最後,雖然我們的專題還有可以再改進的地方,但我認為至少我們的目的都有達成,希望透過未來一步的優化,之後能真正發布上線,幫助帕金森氏症的患者。
